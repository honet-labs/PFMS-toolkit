## 🎨 Theme Overview

### Color System
- **Primary**: Deep blue (#1e3a8a) for main actions
- **Secondary**: Teal (#0d9488) for alternatives
- **Status**: Green, amber, red, cyan for states
- **Neutral**: Gray scale 50-900

### Components
- Navbar with gradient logo
- Icon-integrated forms
- Professional cards with hover effects
- Data tables with status badges
- Color-coded buttons and alerts
- Stat cards for metrics
- Progress bars and indicators

### Responsive Breakpoints
- Desktop: ≥992px
- Tablet: 768-991px
- Mobile: <768px
- Small mobile: <576px

## 🐛 Support & Troubleshooting

### Common Issues

**Gradient text not showing**
→ See "Issue: Gradient text not showing" in UI_THEME_DOCUMENTATION.md

**Icons not displaying**
→ Check Font Awesome CDN link in layouts/app.php

**Select-all not working**
→ Ensure checkboxes have correct IDs/classes

**Mobile layout broken**
→ Check viewport meta tag in layouts/app.php

### Get Help
1. Check relevant documentation file
2. Search for issue in UI_THEME_DOCUMENTATION.md
3. Review THEME_CHANGELOG.md for context
